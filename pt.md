## Anotações para desenvolvedores
"Suas notas em markdown são salvas automaticamente enquanto são escritas. Várias opções de formatação mostradas em tempo real para que você possa conferir exatamente o que escreve. O texto é formatado como você escreve.

Para fragmentos de código, o aplicativo consegue realçar a sintaxe de código em mais de 100 linguagens, incluindo Javascript, Python, HTML e CSS; e você ainda pode colocar múltiplos trechos de código no mesmo fragmento."

## Comunidade de desenvolvimento
O Boostnote possui uma comunidade para programadores. Ela está ganhando muita atenção no GitHub e é mantida por vários contribuidores.
Usado por programadores de mais de 200 países e regiões do mundo.

Você sempre será bem-vindo a nossa comunidade!


## A todo momento; Em qualquer lugar; Para qualquer coisa.
Fácil para escrever anotações de todos os lugares, mesmo se você estiver offline. Quando você atualiza suas anotações no seu laptop, as alterações são refletidas nos seus dispositivos móveis.

Boostnote funciona em todos os lugares.
